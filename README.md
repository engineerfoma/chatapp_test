# Todo List App

Веб-приложение для управления списком задач, построенное на React и Redux Toolkit с использованием упрощенной архитектуры Feature-Sliced Design (FSD).

## Функциональность

- ✅ Добавление новых задач
- ✅ Отметка задач как выполненных
- ✅ Редактирование задач
- ✅ Удаление задач
- ✅ Фильтрация задач (Все / Активные / Выполненные)
- ✅ Сохранение данных в localStorage
- ✅ Drag-and-Drop для изменения порядка задач

## Технологии

- **React 18** - UI библиотека
- **Redux Toolkit** - управление состоянием
- **TypeScript** - типизация
- **Styled Components** - стилизация компонентов
- **Vite** - сборщик и dev-сервер
- **@dnd-kit** - drag-and-drop функциональность

## Установка

```bash
npm install
```

## Запуск

```bash
# Разработка
npm run dev

# Сборка для production
npm run build

# Просмотр production сборки
npm run preview
```

## Структура проекта

Проект организован по принципам упрощенного FSD:

```
src/
├── app/              # Инициализация приложения, store, middleware
├── shared/           # Переиспользуемые модули (утилиты, стили, типы)
├── features/         # Бизнес-фичи (add-todo, filter-todos)
├── entities/         # Бизнес-сущности (todo)
└── widgets/          # Композитные блоки UI (todo-list)
```
